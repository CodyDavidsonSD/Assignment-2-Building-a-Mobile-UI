# Assignment 2

This assignment reacreates 4 screenshots from the Tim Horton's app. The 4 pages are Home, Scan, Discover, and Location. 

## Navigation
The Home, Scan, and Discover pages can be navigated to from the labeled icons on the bottom navbar. 
The Location page can be reached by clicking the large component that says "Order" on the center-left of the Home page.
